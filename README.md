# QA Projects Portfolio

## 📌 О проекте
Репозиторий содержит примеры тестирования мобильных и веб-приложений.

Проекты выполнены в рамках практики и включают чек-листы, баг-репорты и анализ дефектов.

---

## 📱 Mobile Testing

📁 /mobile-testing/

Содержит:
- Регрессионный чек-лист
- Баг-репорты
- Связи тестов и дефектов

📄 Основной файл:
mobile_testing.xlsx

Используемые инструменты:
- Android Studio (эмуляция устройств, тестирование на Android)
- Тестирование на разных разрешениях экрана и версиях ОС

---

## 🌐 Web Testing

📁 /web-testing/

Содержит:

- Проверку пользовательских сценариев
- Найденные дефекты

📄 Основной файл:
web_testing.xlsx
