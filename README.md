# QA Projects Portfolio

## 📌 О проекте
Репозиторий содержит примеры тестирования мобильных и веб-приложений.

Проекты включают чек-листы и баг-репорты, выполненные в рамках практики.

---

## 📱 Mobile Testing

📁 /mobile-testing/

Содержит:
- Регрессионный чек-лист
- Баг-репорты
- Связи тестов и дефектов

📄 Основной файл:
mobile_testing.xlsx

Использовалось:
- Android Studio (эмуляция устройства, тестирование на Android)

---

## 🌐 Web Testing

📁 /web-testing/

Содержит:
- Чек-лист тестирования веб-приложения
- Проверку пользовательских сценариев
- Найденные дефекты

📄 Основной файл:
web_testing.xlsx

Использовалось:
- Chrome DevTools (анализ DOM, проверка запросов, работа с сетью)
